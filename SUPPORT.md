# Support

Use GitHub Issues for bug reports, feature requests, and documentation
problems. Use the `question` label for general usage questions unless GitHub
Discussions is enabled for the repository.

Security reports must follow `SECURITY.md`; do not open public issues for
vulnerabilities or leaked secrets.

Before asking for help, include:

- the `revrem --version` output;
- the command you ran;
- the target repository type;
- the relevant `.revrem/runs/<timestamp>/summary.json` excerpt with secrets
  removed;
- the verification command that failed, if any.
